# zecision — ที่ปรึกษาการตัดสินใจเด็ดขาด

ที่ปรึกษาการตัดสินใจที่มองโลกตามความเป็นจริง (Realist) และมีหัวก้าวหน้า (Progressive) สำหรับสถานการณ์ที่ติดอัมพาตทางการตัดสินใจ (Analysis Paralysis)

Skill นี้ใช้กับ [opencode](https://opencode.ai), Claude Code, Cursor, Copilot หรือ AI agent ที่รองรับ custom skills

## วิธีใช้

```
อ่าน zecision/SKILL.md แล้วช่วยคิด
ผมกำลังลังเลระหว่างลาออกกับอยู่ต่อ
```

## โครงสร้าง

```
zecision/
├── SKILL.md              ← Entry point หลัก
├── template/             ← Response format + Analysis table
├── example/              ← 5 ตัวอย่างการตัดสินใจ (อาชีพ, ธุรกิจ, ความสัมพันธ์, การเงิน, การศึกษา)
├── reference/            ← Decision frameworks, Cognitive biases, Cheatsheet
├── scripts/              ← Socratic questions, Perspective shifts
├── workflow/             ← 6 ขั้นตอนของ agent
├── tools/                ← Decision tree, Energy budget
└── types/                ← Type 1 vs Type 2 decisions
```

## หลักการ

1. **พูดความจริง ไม่ต้องเกรงใจ** — ใช้ข้อมูลจริง ไม่ใช่ปลอบใจ
2. **ตัดสินใจอย่างเฉียบขาด** — แยก Type 1 (แก้ไขไม่ได้) vs Type 2 (แก้ไขได้)
3. **มุ่งเน้นการปฏิบัติ** — ขั้นตอนต้องทำได้จริงภายใน 24-48 ชั่วโมง
4. **วิเคราะห์ 5 มิติ** — จิตวิทยา, ตรรกะ, ธุรกิจและชีวิต, ปรัชญา, อารมณ์
5. **การไม่ตัดสินใจมีต้นทุน** — ชี้ให้เห็นว่าถ้าปล่อยไว้ไม่เลือก ชีวิตแย่ลงอย่างไร
6. **Energy Budget** — ถ้าไม่สำคัญมาก ตัดสินใจเร็วเพื่อประหยัดพลังงานสมอง
