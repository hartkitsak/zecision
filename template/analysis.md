# ตารางวิเคราะห์ 5 มิติ (5-D Analysis Framework)

ใช้ Framework นี้ในการวิเคราะห์ทุกการตัดสินใจ

---

## มิติที่ 1: จิตวิทยา (Psychology)

| คำถามหลัก | สิ่งที่ต้องวิเคราะห์ |
|-----------|-------------------|
| ทางเลือกนี้ส่งผลต่อความมั่นใจอย่างไร? | Self-esteem, Growth mindset |
| ทางเลือกนี้สร้างความเครียดระยะยาวไหม? | Chronic stress, Burnout risk |
| ทางเลือกนี้สอดคล้องกับตัวตนที่อยากเป็นไหม? | Identity congruence |
| มี Cognitive Bias อะไรแอบแฝงอยู่? | เปิด `reference/cognitive-biases.md` |

---

## มิติที่ 2: ตรรกะ (Logic)

| คำถามหลัก | สิ่งที่ต้องวิเคราะห์ |
|-----------|-------------------|
| มีข้อมูลอะไรสนับสนุน? | Data, Evidence, Research |
| ความน่าจะเป็นของความสำเร็จ? | Probability, Statistics |
| ข้อสันนิษฐานอะไรที่ยังไม่ถูกพิสูจน์? | Assumptions, Blind spots |
| มี Framework การตัดสินใจอะไรช่วยได้? | เปิด `reference/decision-frameworks.md` |

---

## มิติที่ 3: ธุรกิจและชีวิต (Business & Life)

| คำถามหลัก | สิ่งที่ต้องวิเคราะห์ |
|-----------|-------------------|
| ผลกระทบทางการเงินระยะสั้น/ยาว? | Cash flow, ROI, Opportunity cost |
| ผลกระทบต่ออาชีพการงาน? | Career trajectory, Network, Skills |
| ผลกระทบต่อความสัมพันธ์? | Family, Partner, Friends |
| ผลกระทบต่อสุขภาพ? | Physical health, Sleep, Lifestyle |

---

## มิติที่ 4: ปรัชญา (Philosophy)

| คำถามหลัก | สิ่งที่ต้องวิเคราะห์ |
|-----------|-------------------|
| ทางเลือกนี้สอดคล้องกับค่านิยมหลักไหม? | Core values, Morals |
| อีก 10 ปี มองกลับมา จะเสียใจไหม? | Regret Minimization |
| ทางเลือกนี้ทำให้ชีวิตมีความหมายขึ้นไหม? | Purpose, Meaning |
| กำลังตัดสินใจจากความกลัวหรือความรัก? | Fear vs Love framework |

---

## มิติที่ 5: อารมณ์ (Emotion)

| คำถามหลัก | สิ่งที่ต้องวิเคราะห์ |
|-----------|-------------------|
| สัญชาตญาณบอกอะไร? | Gut feeling, Intuition |
| อารมณ์ตอนนี้กำลังบดบังเหตุผลไหม? | Emotional hijack |
| ถ้าตื่นมาพรุ่งนี้เลือกแล้ว จะรู้สึกอย่างไร? | Morning-after test |
| ถ้าไม่มีใครรู้ จะเลือกอะไร? | Anonymous choice test |
