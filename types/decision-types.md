# Decision Types — Type 1 vs Type 2

## Type 1: คิดหนัก (One-Way Door)
- พลิกกลับไม่ได้/ยาก, ต้นทุนสูง
- **ตัวอย่าง**: ซื้อบ้าน, แต่งงาน, มีลูก, ลาออก, ย้ายประเทศ
- **วิธี**: Pre-Mortem → Risk Mitigation → Consult → Sleep On It → Soft Landing

## Type 2: ลงมือทันที (Two-Way Door)
- พลิกกลับง่าย, ต้นทุนต่ำ
- **ตัวอย่าง**: กินอะไร, ซื้อของ, เรียนคอร์สสั้น, เลือกเครื่องมือ
- **วิธี**: Fail-Fast Protocol → ตั้งเวลา 15 นาที → ลองก่อน → เขียน Exit Plan

## Type 1.5: คิดหนักปานกลาง
- พลิกกลับได้แต่มีต้นทุน
- **ตัวอย่าง**: เปลี่ยนสายอาชีพ, ย้ายเมือง, เรียน ป.โท, เริ่มธุรกิจเล็ก
- **วิธี**: Analysis Lite → กำหนดเงื่อนไข (ถ้า 3 เดือนไม่ดี→เปลี่ยน) → Try Before Commit

## Matrix เปรียบเทียบ
| | Type 1 | Type 1.5 | Type 2 |
|--|--------|---------|--------|
| Reversibility | ต่ำมาก | ปานกลาง | สูง |
| Cost of Error | สูงมาก | ปานกลาง | ต่ำ |
| เวลาที่ควรคิด | วัน-สัปดาห์ | ชม.-วัน | นาที-ชม. |
| เครื่องมือ | Pre-Mortem, Full 5-D | Lite 5-D, Trial | Fail-Fast, สุ่ม |
| ข้อควรระวัง | Analysis Paralysis | Overthinking | Impulsive Decision |

**จำง่าย**: Type 1 = One-Way Door → คิดก่อนเดิน / Type 2 = Two-Way Door → เดินก่อนคิด (ผิดก็กลับมา)

**คำเตือน**: Type 2 ที่ทำบ่อยๆ สะสมกลายเป็น Type 1 ได้ (กินข้าวนอกบ้านทุกวัน→สุขภาพ, ซื้อของออนไลน์ทุกอาทิตย์→การเงิน)
