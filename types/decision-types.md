# Decision Types — Type 1 vs Type 2

อ้างอิงแนวคิดจาก Jeff Bezos และหลักการตัดสินใจเชิงกลยุทธ์

---

## Type 1: คิดหนัก (One-Way Door)

### ลักษณะ
- **พลิกกลับไม่ได้ หรือพลิกกลับได้ยากมาก**
- **ต้นทุนสูง** ถ้าเลือกผิด
- ต้องใช้การวิเคราะห์ที่ลึก และรอบคอบ

### ตัวอย่าง
| เรื่อง | ตัวอย่าง |
|-------|---------|
| บ้าน | ซื้อบ้าน, ซื้อคอนโด (ผูกพัน 30 ปี) |
| การแต่งงาน | แต่งงาน, หย่าร้าง |
| ลูก | การตัดสินใจมีลูก, วิธีการเลี้ยงดู |
| อาชีพ | ลาออกจากงาน (เสียรายได้ + เส้นทางอาชีพ) |
| การเงินใหญ่ | ขายบ้าน, ลงทุนก้อนใหญ่ |
| ย้ายประเทศ | การย้ายถิ่นฐานถาวร |

### วิธีจัดการ
```
1. Pre-Mortem: สมมติว่าเลือกผิด — แย่ที่สุดคืออะไร?
2. Risk Mitigation: เตรียมแผนรับมือ
3. Consult: หาความเห็นจากผู้มีประสบการณ์
4. Sleep On It: อย่าตัดสินใจตอนอารมณ์ร้อน
5. Soft Landing: มีทางออกฉุกเฉินเสมอ
```

---

## Type 2: ลงมือทันที (Two-Way Door)

### ลักษณะ
- **พลิกกลับได้ง่าย**
- **ต้นทุนต่ำ** ถ้าเลือกผิด
- ควรตัดสินใจเร็ว ไม่ต้องคิดมาก

### ตัวอย่าง
| เรื่อง | ตัวอย่าง |
|-------|---------|
| ชีวิตประจำวัน | กินอะไร, ใส่เสื้อตัวไหน, ดูหนังเรื่องไหน |
| ซื้อของ | ซื้อของใช้, ซื้ออุปกรณ์ |
| การลอง | ทดลอง Methods, เรียนคอร์สสั้น |
| งานย่อย | เลือกเครื่องมือทำงาน, จัดโต๊ะใหม่ |
| Social | จะตอบแชทอย่างไร, จะไปงานเลี้ยงไหม |

### วิธีจัดการ
```
1. Fail-Fast Protocol: รู้ว่าสัญญาณเตือนคืออะไร
2. ตั้งเวลา: ให้เวลาตัดสินใจไม่เกิน 15 นาที
3. ลองก่อน: ถ้าผิดก็เปลี่ยน
4. เขียน Exit Plan: รู้ว่าถ้าใช้ไม่ได้ผลจะเลิกอย่างไร
5. อย่าคิดมาก: Energy Budget มีไว้ใช้กับเรื่องสำคัญ
```

---

## Type 1.5: คิดหนักปานกลาง (Middle Ground)

### ลักษณะ
- พลิกกลับได้ แต่มีต้นทุนในการพลิก
- ต้องการการวิเคราะห์แบบย่อ

### ตัวอย่าง
- เปลี่ยนสายอาชีพ
- ย้ายเมือง
- เรียนปริญญาโท
- เริ่มธุรกิจขนาดเล็ก

### วิธีจัดการ
```
1. Analysis Lite: ใช้ 5 มิติ แต่ไม่ต้องละเอียดเท่า Type 1
2. กำหนดเงื่อนไข: "ถ้าผ่าน 3 เดือนแล้วไม่ดี → เปลี่ยน"
3. Try Before Commit: ทดลองก่อนผูกมัดเต็มตัว
```

## Matrix เปรียบเทียบ

| คุณสมบัติ | Type 1 | Type 1.5 | Type 2 |
|----------|--------|---------|--------|
| Reversibility | ต่ำมาก | ปานกลาง | สูง |
| Cost of Error | สูงมาก | ปานกลาง | ต่ำ |
| เวลาที่ควรใช้คิด | วัน - สัปดาห์ | ชม. - วัน | นาที - ชม. |
| เครื่องมือ | Pre-Mortem, Full 5-D | Lite 5-D, Trial | Fail-Fast, สุ่ม |
| ข้อควรระวัง | Analysis Paralysis | Overthinking | Impulsive Decision |
| Response | ชี้ขาดแบบมี Plan | ชี้ขาดแบบมีทางเลือก | ชี้ขาดแบบเร่งด่วน |

---

## หลักการจำ

```
Type 1 = One-Way Door → คิดก่อนเดิน
Type 2 = Two-Way Door → เดินก่อนคิด (ผิดก็กลับมา)
Type 1.5 = Revolving Door → เดินแต่เตรียมกลับ
```

---

## กรณีพิเศษ: เมื่อ Type 2 กลายเป็น Type 1

| การตัดสินใจเล็ก | ถ้าทำซ้ำๆ กลายเป็น |
|----------------|-------------------|
| กินข้าวนอกบ้านทุกวัน | สุขภาพ - Type 1 |
| ซื้อของออนไลน์ทุกอาทิตย์ | การเงิน - Type 1 |
| ตอบแชทไม่คิดทุกครั้ง | ความสัมพันธ์ - Type 1 |

> **คำเตือน**: Type 2 ที่ทำบ่อยๆ สะสมกลายเป็น Type 1 ได้เสมอ
