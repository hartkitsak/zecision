# Decision Tree — แยก Type 1 vs Type 2

## Step 1: Reversibility (พลิกกลับได้ไหม?)
```
├── พลิกกลับยาก/ไม่ได้ → Type 1 (ซื้อบ้าน, แต่งงาน, ลาออก)
├── พลิกได้แต่มีต้นทุน → Type 1.5 (เปลี่ยนอาชีพ, ย้ายเมือง)
└── พลิกกลับง่าย → Type 2 (ซื้อของ, เรียนคอร์สสั้น)
```

## Step 2: Cost of Failure (ต้นทุนถ้าพลาด)
```
├── สูงมาก → Type 1 (เสียเงินมาก, เวลา >1 ปี)
├── ปานกลาง → Type 1.5 (เสียเงินบ้าง, เวลา 1-6 เดือน)
└── ต่ำ → Type 2 (เสียเงินนิดหน่อย, เวลา <1 เดือน)
```

## Step 3: Energy Budget
```
├── พลังสูง → ใช้ทุกเครื่องมือ
├── ปานกลาง → 5-D + Action
└── ต่ำ → พักก่อน (ถ้าสำคัญ) หรือสุ่ม (ถ้าไม่สำคัญ)
```

## Decision Matrix
| | ต้นทุนต่ำ | ต้นทุนกลาง | ต้นทุนสูง |
|--|---------|-----------|---------|
| **พลิกง่าย** | Type 2 | Type 2 | Type 1.5 |
| **พลิกยาก** | Type 1.5 | Type 1 | Type 1 |
