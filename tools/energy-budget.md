# Energy Budget Calculator

## สูตร
`Energy Score = (Importance × 2) + (Urgency × 1.5) - (Fatigue × 1)`

| ค่า | ระดับ |
|-----|-------|
| **Importance** | 0-3 เล็กน้อย / 4-6 ปานกลาง / 7-8 สำคัญ / 9-10 ชีวิต |
| **Urgency** | 0-3 ไม่เร่ง / 4-6 ปานกลาง / 7-8 เร่ง / 9-10 ฉุกเฉิน |
| **Fatigue** | 0-3 สมองสด / 4-6 เพลีย / 7-8 เหนื่อย / 9-10 Burnout |

## เกณฑ์
| Score | คำแนะนำ |
|-------|--------|
| > 15 | พลังสูง — ใช้ทุกเครื่องมือ |
| 10-15 | ปานกลาง — ใช้ 5-D + Action |
| 5-10 | ต่ำ — Decision Tree ถ้าสำคัญ, ไม่สำคัญ→สุ่ม |
| < 5 | หมด — พักก่อน! อย่าตัดสินใจ |
